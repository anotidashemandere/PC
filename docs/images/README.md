Put **screenshots and photos for your project documentation** here.

These files are **not used by the running application** — they are for reports, presentations, and README illustrations only.

## Tips

- Use PNG format for crisp UI screenshots
- Recommended width: 1280px or 1920px
- Name files with numbers so they sort in order
- Do not put applicant CVs or private data in this folder

## App images vs documentation images

| Folder | Purpose |
|--------|---------|
| `static/` | Images used **by the website** (logo fallback, banners) |
| `uploads/branding/` | Logo and video uploaded **via HR Settings** |
| `docs/images/` | Screenshots **for your documentation only** |

See `PROJECT_DOCUMENTATION.md` for full details on where to put logos, videos, and media.
